✨ 功能概述
白板应用新增了完整的文字对齐功能，支持水平和垂直两个方向的对齐设置。 用户可以通过右键点击卡片，在颜色选择器中设置文字对齐方式。

📋 使用方法
右键点击任意卡片，打开 颜色选择器
在颜色选择器中找到 "文字对齐" 部分
选择 水平对齐：左对齐 ⬅️、居中 ↔️、右对齐 ➡️
选择 垂直对齐：顶部 ⬆️、居中 ↕️、底部 ⬇️
点击 "完全居中" 按钮可快速设置水平和垂直都居中
🔧 技术实现
核心属性：
• textAlign: 'left' | 'center' | 'right' - 水平对齐
• textVerticalAlign: 'top' | 'center' | 'bottom' - 垂直对齐

样式应用：
• 使用 CSS Flexbox 实现垂直对齐
• 使用 CSS text-align 实现水平对齐
• 支持实时预览和即时生效

兼容性：
• 支持正面和背面编辑
• 支持富文本和Markdown模式
• 与现有功能完全兼容
